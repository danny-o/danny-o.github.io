An android app template for an online store

The app particularly deals with clothes but can be customized to ones needs.

Features

The Login Page

One can login via email or with google. To login with email you must have created an account

Product details page

Get more details about a product and add it to favorites or to cart

Search for items

Search for specific items in the products page either by a specific name or category

The checkout page

Edit the quantity required, remove products from the cart and view subtotals and totals

Other Features

Cart reminder

Get notified if you leave items pending in cart.

Add products

Add products to the catalogue if you are an admin.

Sorting options

Display only the products that have been favorited.
